掘金编辑器 ​

本人编写了一个 Typecho 上使用的文章编辑器，用于与 JJ 主题实现更深度化的体验。

编辑器地址：Typecho_Plugin_JJEditor

编辑器本身支持数学公式和图表，具体请自行查看编辑器的文档说明。

编辑器预览 ​

联动 JJ 主题友链页样式 ​

在独立页编辑页面时，底部会有一个是否启用友链样式的选项，勾选后，会启用 JJ 主题的友链样式，需要注意的是，你最好只在友链页面启用该功能，否则会影响到其他页面的样式。
